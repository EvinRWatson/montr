# MONTR
An extremely lightweght web-service for system resource and diagnostic data. Designed to be used for dashboard monitoring and automation tasks.

## Data served on request:

### CPU:
- Usage
- Tempurature
- Cores

### Memory:
- Capacity
- Usage 
- Speed

### Disk:
- Capacity
- Usage
- Speed 
- S.M.A.R.T. Status
